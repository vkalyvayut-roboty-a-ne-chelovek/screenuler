# screenuler
Simple Python Tkinter screen ruler

Keybindings are:

```CTRL-Q``` or ```ESC``` - exit

```CTRL-S``` - change size to small (250x75)

```CTRL-M``` - change size to medium (500x75)

```CTRL-L``` - change size to large (750x75)

```Left arrow```, ```Right arrow```, ```Up arrow```, ```Down arrow``` - move
